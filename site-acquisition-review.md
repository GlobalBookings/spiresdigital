# Whole-site SEO and agency-acquisition review

Reviewed: 18 July 2026

## Outcome

The site now has a coherent ecommerce agency proposition, useful ungated tools, commercial service journeys, selected evidence and accountable authorship. Its technical foundations are suitable for crawling and indexation. The largest remaining constraint is authority and measured post-launch demand—not missing page volume.

## Evidence used

- 16-month Search Console export supplied for `spiresdigital.com`
- Rendered production build and sitemap inventory
- Current Google Search Central guidance on useful content, title links, structured data and performance measurement
- Desktop and mobile journey checks
- Current search-result review for ecommerce profit calculators, Google Ads audits and UK ecommerce agencies

The Search Console query export contains 1,000 rows, 51,002 impressions and 36 clicks. These are the visible top-query rows, not a complete property total. The page export contains 296 rows and must not be added to the query table because Search Console aggregation differs.

## Remaining risks and next actions

1. **Authority:** useful tools at average positions around 80–95 will not recover through on-page changes alone. Execute the digital PR and partnership plan with original data, workshops and relevant editorial contributions.
2. **Live account verification:** Search Console ownership, sitemap status and post-launch indexing still require access to the property.
3. **Post-launch sample:** wait for four complete weeks before declaring conversion or ranking improvement.
4. **Content maintenance:** 102 indexable articles are an asset only if current. Refresh by actual impressions and commercial relevance; consolidate rather than add calendar filler.
5. **Proof depth:** named references and retention-specific evidence remain permission-dependent.
6. **Lead quality:** record qualified consultations and accepted work outside GA4 so the site is not optimised towards low-value form volume.

## Publishing rule for new content

Publish only when the proposed page has all five:

1. A distinct search or buyer job not already served by an existing page.
2. Original experience, a calculation, a tool, a decision framework or a source-led update.
3. A maintained owner and review trigger.
4. A natural next step into another useful resource or relevant service.
5. A measurement plan that distinguishes reach, useful engagement and qualified commercial outcome.

If those conditions are not met, improve an existing page instead.
